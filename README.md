## 📖 Project Overview

This project presents an **AI-based Electric Vehicle Drive Mode Prediction System** that intelligently predicts the most suitable driving mode based on vehicle parameters and driving conditions.

The system leverages **Machine Learning techniques** to analyze vehicle-related inputs and recommend the optimal driving mode, improving:

- ⚡ Energy efficiency
- 🔋 Battery utilization
- 🚗 Driving performance
- 🌱 Sustainable mobility

---

## ✨ Features

✅ Smart drive mode prediction using Machine Learning

✅ Interactive web-based user interface

✅ Real-time prediction capability

✅ Trained ML model for intelligent decision making

✅ Data-driven mode selection

✅ User-friendly and responsive design

---

## 🧠 How It Works

1. User enters vehicle/driving parameters.
2. Input data is processed.
3. Trained Machine Learning model analyzes the inputs.
4. The system predicts the most suitable EV drive mode.
5. Results are displayed through the web interface.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/dasaridevika/project_stage2.github.io.git
```

### Navigate to the Project Directory

```bash
cd project_stage2.github.io
```

### Install Required Libraries

```bash
pip install pandas numpy scikit-learn flask
```

### Run the Prediction Script

```bash
python predict_mode.py
```

### Train the Model

```bash
python train_mode_model.py
```
